# Phishing Email Analyzer (Offline)

## Overview
This project is a simple offline tool designed to analyze phishing emails 
for security awareness and incident response purposes.

## Why this project?
Phishing remains one of the most common attack vectors.
This tool demonstrates practical security fundamentals aligned with
Security+ and GRC principles.

## Features
- Email header analysis
- URL and domain extraction
- Basic risk indicators

## Disclaimer
For educational and defensive purposes only.
